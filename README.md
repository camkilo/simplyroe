
# Realm of Echoes — Prototype (replace name)

One-line: A next-gen, chemistry-grounded crafting MMORPG built for infinite emergent systems.

## Why this repo
Prototype, experiment, and ship fast. All assets, demo, and docs here. Designed to deploy from GitHub → Render (backend) + Vercel/Render (frontend).

## Quick start (for contributors)
1. `git clone <repo>`
2. `cd repo`
3. `cd api && pip install -r requirements.txt` (or `npm install`)
4. `cd app && npm install && npm run dev`
5. Visit http://localhost:3000 and http://localhost:8000

## What to edit first
- `docs/vision.md` — core mechanics & crafting rules
- `app/src/components/Shared` — UI pieces to iterate fast
- `api/blueprint_engine.py` — deterministic blueprint discovery
